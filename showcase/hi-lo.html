2D Hi-Lo Game in Unity

Overview

Hi-Lo is my first ever Unity game! Jump in and test your luck by guessing if the next card will be higher or lower.

While it's missing fancy animations and cool sound effects (turns out game dev takes longer than I thought), the game works and you can challenge yourself to beat your high score.

I've since moved on to bigger projects, but this little card game will always have a special place in my heart as my first step into game development. Give it a try and see how many correct guesses you can make!

Technologies Used

  • Unity Engine
  • C# Programming

Key Features

  • Real-time Score Tracking
  • Instant Win/Lose Feedback
  • Very Simple UI
